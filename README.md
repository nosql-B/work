# nosql大作业

## 一、项目概括
实现简易的新闻网站，使用Java开发，涉及mysql，redis，以及MongoDB三大主流数据库。
## 二、技术栈
前端:bootstrap,Echarts,Ajax,boostrap-table,bootstrap-fileinput,bootbox  
前端模板引擎:freemarker  
后端:
1. 核心框架:Springboot
2. 持久层框架:Mybatis
3. 分页插件:PageHelper
4. 持久层插件:mybatis-generator
5. 项目构建:Maven
6. JDK版本:1.8


三大主流数据库：
1. Mysql
2. MongoDB
3. Redis

## 三、项目内容
1. 数据库新闻信息表设计
2. 登录注册注销
3. 新闻前台
4. 新闻后台
5. MongoDB设计评论数据存储
6. 将热点新闻缓存至redis数据库以便数据及时更新
7. 数据观星功能
