# nosql大作业

## 一、项目概括
实现简易的新闻网站，使用Java开发，涉及mysql，nosql，以及MongoDB三大主流数据库。
## 二、技术栈
前端框架：bootstrap
前端模板引擎:freemarker
后端:Springboot+Mybatis


三大主流数据库：
1. Mysql
2. MongoDB
3. Redis

## 三、项目内容
1. 数据库新闻信息表设计
2. 新闻前台页面实现
3. 新闻后台页面实现
4. MongoDB设计评论数据存储
5. 将热点新闻缓存至redis数据库以便数据及时更新

## 四、项目进度

**相关工具的测试准备：**
1. SpringMVC相关页面跳转
2. News的Mybatis映射
3. Mongodb的连接测试

**项目内容进度**

1. Mysql数据库的新闻表设计
2. 新闻表的相关Mybatis实现增删改查
3. 新闻前台的前端demo以及相关路由实现，实现前后端交互
4. 登录注册注销等功能实现。
5. 前端新闻的评论功能
