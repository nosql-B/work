# nosql大作业

## 一、项目概括
实现简易的新闻网站，使用python开发，涉及mysql，nosql，以及MongoDB三大主流数据库。
## 二、技术栈
前端框架：bootstrap

python相关库：
1. ORM(SQLALchemy),pymysql 
2. flask,flask_wtf
3. Mongoengine

三大主流数据库：
1. Mysql
2. MongoDB
3. Redis

## 三、项目内容
1. 数据库新闻信息表设计
2. 新闻前台页面实现
3. 新闻后台页面实现
4. MongoDB设计评论数据存储
5. 将热点新闻缓存至redis数据库以便数据及时更新
